# Filtr Kalmana dla Silnika JGA25-370

Projekt z przedmiotu Wprowadzenie do Sztucznej Inteligencji realizujący estymację prędkości silnika DC z wykorzystaniem filtru Kalmana. System porównuje skuteczność filtracji probabilistycznej z klasycznym filtrem uśredniającym SMA, uwzględniając opóźnienie transportowe i szum pomiarowy enkodera.
